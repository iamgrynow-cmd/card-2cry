Frequently Asked Questions

Everything you need to know about NexPay cards, plans, dashboard and API access.

What is NexPay?
NexPay is a premium crypto-card platform concept for virtual cards, crypto-powered online spending, business payments and API access.
How do I create a card?
Create an account, login, choose a plan from pricing, and your selected card plan will appear in your dashboard.
Can I add funds?
The demo dashboard can support add funds flow using local storage. For real payment handling, backend and compliance infrastructure are required.
Do you support physical cards?
Physical cards are presented as a premium upcoming feature with waitlist style onboarding.
Is KYC required for real cards?
Yes. Any real card issuing or financial service would require KYC, compliance review, issuer approval and regional eligibility checks.
Can businesses request API access?
Yes. Businesses, agencies and fintech teams can use the contact page to request API or partnership access.
